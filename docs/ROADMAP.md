# Roadmap

O.R.A.C.L.E Framework is packaged as a validated defensive cybersecurity framework with a clear extension roadmap.

## Completed

- Core framework.
- Oracle GUI.
- MutantShield detection.
- QAuthCore assurance.
- EthicQ governance.
- ChronoLedger audit.
- GhostTunnel secure response.
- Evolution Engine.
- CSE adaptation.
- DoHBrw anomaly adapter.
- Module capability validation.
- GitHub release.

## Deferred / Future

- GAN synthetic threat generation.
- LSTM/GNN candidate-safe retraining execution after contracts are satisfied.
- SIEM/SOAR/EDR integrations.
- Production deployment hardening.
- Cloud deployment.
- Container orchestration.
- Model artifact packaging through Git LFS or GitHub Release assets if public redistribution is approved.
- Additional domain adapters for unvalidated traffic families.
- Expanded API authentication and multi-tenant deployment controls.

## SIEM/SOAR/EDR Integration Roadmap

- Phase 1: Wazuh SIEM JSON/syslog connector.
- Phase 2: Shuffle SOAR webhook.
- Phase 3: osquery, Sysmon, or Velociraptor EDR telemetry adapter.

See `docs/SIEM_SOAR_EDR_INTEGRATION.md` for planned architecture and security controls.

## Deferred By Design

- GAN training is not part of the current release.
- SIEM/SOAR/EDR integrations are not part of the current release.
- Automatic production promotion is blocked by design.
