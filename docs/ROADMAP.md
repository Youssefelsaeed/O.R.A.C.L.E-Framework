# Roadmap

O.R.A.C.L.E Framework is packaged as a validated defensive cybersecurity framework with a clear extension roadmap.

## Planned Work

- Production deployment hardening profiles.
- Optional SIEM/SOAR/EDR integrations.
- Quality-gated GAN synthetic data generation.
- Candidate-safe LSTM/GNN retraining when sequence and graph contracts are satisfied.
- Model artifact packaging through Git LFS or GitHub Release assets if public redistribution is approved.
- Additional domain adapters for unvalidated traffic families.
- Expanded API authentication and multi-tenant deployment controls.

## Deferred By Design

- GAN training is not part of the current release.
- SIEM/SOAR/EDR integrations are not part of the current release.
- Automatic production promotion is blocked by design.
